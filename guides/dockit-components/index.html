What is Dockit?
Dockit is a modern documentation theme built on Astro Starlight with custom components and enhanced UI.
Can I customize the theme?
Yes! Dockit uses Tailwind CSS and Starlight's theming system, making it highly customizable.
How do I add new pages?
Simply create new .md or .mdx files in the src/content/docs/ directory.

Component Props

Section titled “Component Props”

Grid

Section titled “Grid”
  • columns - Number of columns (default: 2)

NewCard

Section titled “NewCard”
  • title - Card title
  • icon - Starlight icon name
  • size - “large” (default) or “sm”

Button

Section titled “Button”
  • label - Button text
  • link - Destination URL
  • variant - “primary” (default), “secondary”, or “outline”

Accordion

Section titled “Accordion”
